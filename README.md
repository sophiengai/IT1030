# IT1030SEF GROUP PROJECT 

### Guide for running the system
1. Download all files
2. Start from running the ***HomePage.html*** file

[On ***LostPage/ FoundPage*** you can: ]
- Review the uploaded items
- Sort. filter and search the display list

[On ***LostItemInput/ FoundItemInput page*** you can: ]
- Upload photo of the item
- Type-in **discription, location, lost/ found date & time, contact, remark** and choose the **item type** with the check box

[Redirecting between different pages]

We used the button function to manage the flow which connect between different pages

- On the top of the ***Home page***, we placed two buttons that serve as the route to the ***Found page*** and the ***Lost page***
- You can switch between the ***Lost/ Found page*** and the ***LostItemInput/ FoundItemInput page***, as well as return to the ***Home page*** from either page.
